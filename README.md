# NCK Calculator (Single Page)

A simple, clean daily calculator with:

- Unit Management (4 fields + Total Unit)
- Cash Management (6 fields + Total Cash)
- Calculations & Deductions (Total Unit − manual, Total Cash − manual)

## Run

Option 1 (simplest):
- Double-click `index.html` to open it in your browser.

Option 2 (recommended local server):

```bash
cd "/Users/kyawzin001/Desktop/NCK Calculator"
python3 -m http.server 5173
```

Then open `http://localhost:5173`.
